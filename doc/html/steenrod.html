NAME

Steenrod - Tcl package for Steenrod algebra computations

SYNOPSIS

package require Steenrod 1.0
matrix subcommand ?args?
poly subcommand ?args?

DESCRIPTION

The Steenrod package provides a dynamically loadable extension to the Tcl language, which enables Tcl to deal efficiently with Steenrod algebra computations.
